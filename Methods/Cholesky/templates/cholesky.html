The Cholesky decomposition is a numerical algorithm for solving a system of linear equations \( A \cdot x = b \) by factorizing the matrix \( A \) into a lower triangular matrix \( L \) and its transpose \( L^T \).

{% endblock %} {% block help %}

To use the Cholesky decomposition method:

  • Provide a symmetric and positive-definite square matrix \( A \) and a right-hand side vector \( b \).
  • Specify a tolerance and a maximum number of iterations for error-based convergence analysis (optional).
{% endblock %} {% block syntax %}

The general syntax for the Cholesky decomposition method is:

cholesky_decomposition(A, b, tol, max_iter)

Where:

  • A: Symmetric and positive-definite coefficient matrix (size \( n \times n \)).
  • b: Right-hand side vector.
  • tol: Tolerance for stopping criteria (optional).
  • max_iter: Maximum number of iterations (optional).
{% endblock %} {% block examples %}

Example: To solve a system of linear equations:

Matrix A:
| 25   15  -5 |
| 15   18   0 |
| -5    0  11 |

Vector b: [1, 1, 1]
Tolerance: 0.0001
Max Iterations: 100
