The Gauss method is a numerical algorithm for solving a system of linear equations \( A \cdot x = b \) by using forward elimination and back substitution.

{% endblock %} {% block help %}

To use the Gauss method:

  • Provide a square coefficient matrix \( A \) and a right-hand side vector \( b \).
  • Specify a tolerance and maximum iterations (if using error-based convergence).
{% endblock %} {% block syntax %}

The general syntax for the Gauss method is:

gauss_elimination(A, b, tol, max_iter)

Where:

  • A: Coefficient matrix (size \( n \times n \)).
  • b: Right-hand side vector.
  • tol: Tolerance for error-based stopping criteria (optional).
  • max_iter: Maximum number of iterations (optional).
{% endblock %} {% block examples %}

Example: To solve a system of linear equations:

Matrix A:
| 10  -1   2 |
| -1  11  -1 |
|  2  -1  10 |

Vector b: [6, 25, -11]
Tolerance: 0.0001
Max Iterations: 100
